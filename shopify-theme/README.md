# Premium Shopify Theme (TypeScript)

This theme is rebuilt for your internship submission with a premium visual style and TypeScript-powered frontend logic.

## Stack
- Shopify Liquid (sections/templates/layout)
- CSS (premium responsive design system)
- TypeScript source in `src/theme.ts`
- Compiled storefront script in `assets/theme.js`

## Local build (optional)
1. `npm.cmd install`
2. `npm.cmd run build`

## Upload to Shopify
1. Zip the contents inside `shopify-theme` (not the parent folder).
2. Shopify Admin -> Online Store -> Themes -> Import theme.
3. Publish and customize images/products in Theme Editor.
